{{% callout note %}}
Clique no botão **Slides** acima para ver os slides integrados.
{{% /callout %}}

Slides podem ser adicionados de algumas maneiras:

- **Criar** slides usando o recurso [_Slides_](https://docs.hugoblox.com/reference/content-types/) do Hugo Blox Builder e vincular usando o parâmetro `slides` no front matter do arquivo de palestra
- **Fazer upload** de um deck de slides existente para `static/` e vincular usando o parâmetro `url_slides` no front matter do arquivo de palestra
- **Incorporar** seus slides (por exemplo, Google Slides) ou vídeo de apresentação nesta página usando [shortcodes](https://docs.hugoblox.com/reference/markdown/).

Detalhes adicionais do evento, incluindo [elementos de página](https://docs.hugoblox.com/reference/markdown/) como galerias de imagens, podem ser adicionados ao corpo desta página.
